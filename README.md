## 🧾 Tentang Project

**ALFI-MART POS** adalah aplikasi **Sistem Kasir Online (Point of Sale)** berbasis web yang dibuat untuk membantu pengelolaan transaksi pada toko kelontong secara **digital, cepat, dan efisien**.

Project ini dikembangkan menggunakan **Laravel Framework** dengan database **MySQL**.
